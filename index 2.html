🎓 Education & Learning Experience

My journey in computer science and programming began when I enrolled in my first programming course at university. Over the past two semesters, I have had the privilege of learning two fundamental areas of computer science: Web Programming and C Programming. These courses have not only taught me technical skills but have also shaped the way I think about problem-solving, logic, and creativity in technology.

Web Programming Course (Current Semester): This semester, I have been learning Web Programming, which covers the core technologies used to build websites and web applications. We started with HTML, the foundation of every webpage, learning how to structure content using tags, headings, paragraphs, lists, links, images, and forms. Then we progressed to CSS, where I discovered the power of styling — controlling colors, fonts, layouts, and responsive design. The combination of HTML and CSS gave me the ability to turn simple text documents into visually appealing websites.

We also explored more advanced topics such as using external libraries, working with GitHub for version control, and deploying websites using GitHub Pages. This last skill was particularly exciting because it allowed me to publish my work on the internet for the world to see. I learned how to create a repository, push my code, and enable GitHub Pages to host my website at a public URL. This process taught me about collaborative development workflows used by professional software engineers worldwide.

C Programming Course (Previous Semester): Last semester, I completed the C Programming course, one of the most foundational courses in computer science. C is a low-level, compiled language that gives programmers direct control over memory and hardware, making it an excellent language for understanding how computers actually work. Through this course, I developed skills in variables and data types, control structures (if/else, switch), loops (for, while, do-while), functions, arrays, pointers, and basic file I/O.

I practiced extensively on HackerRank and SoloLearn, two popular online platforms that provided me with structured exercises, instant feedback, and certifications to validate my progress. HackerRank challenges me with algorithm problems, while SoloLearn provided interactive lessons combined with a supportive community of learners from around the world.

My Learning Approach: I believe in learning by doing. Rather than just reading theory, I always try to write code, make mistakes, debug problems, and improve my solutions. Each exercise I completed taught me something new — sometimes a new syntax rule, sometimes a better algorithm, and sometimes a completely different way of thinking about a problem. This hands-on approach has been the cornerstone of my programming education.

Future Goals: Moving forward, I plan to deepen my knowledge of JavaScript to make my websites interactive and dynamic. I am also interested in learning backend web development using frameworks like Node.js or Python/Django, which will allow me to build full-stack web applications. My experience with C has given me a strong foundation in computer science fundamentals, including memory management and algorithmic thinking, which will serve me well throughout my career in software development. I am excited about the endless possibilities that programming offers and look forward to building more complex and impactful projects in the future.
