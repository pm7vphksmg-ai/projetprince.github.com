Exercise 1

Working with Images

Learning how to embed and style images in HTML

🖼️ About This Exercise

In this exercise, I learned how to add images to a web page using the <img> tag, set dimensions, add alt text for accessibility, and link images to other pages.

Key Concepts Practiced

  • The <img> tag with src and alt attributes
  • Setting image width and height in HTML and CSS
  • Using images as hyperlinks with <a><img/></a>
  • Understanding relative vs absolute image paths
  • CSS image styling: border-radius, box-shadow, object-fit
